# Cheesecake Images

Place the exported WordPress images here with the exact filenames listed below.
The website will show an elegant placeholder until the image file is present.

## How to Export from WordPress

1. Log in to your WordPress admin (via server IP or cPanel URL, not the domain)
2. Go to **Media → Library**
3. Download each image, then rename and place it here
